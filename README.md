**READ ME**
1. Clone projectwebtech2 ขึ้นมา
2. ใน terminal ให้พิมพ์ composer install เมื่อ install เสร็จให้พิมพ์ npm install
3. นำไฟล์ .env มาใส่ใน projectwebtech2 จากนั้นเข้าไปแก้ไขไฟล์ php.ini แล้วแก้ไข...
4. database name = project2
   database user = root
   database password =

Email: admin@gmail.com,
Password: 0000
